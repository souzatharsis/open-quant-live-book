1.1 Data Sources

1.1.1 Alpha Vantage

Alpha Vantage offers free access to pricing data including:

  • Stock Time Series Data;
  • Physical and Digital/Crypto Currencies (e.g., Bitcoin);
  • Technical Indicators and
  • Sector Performances.

The data are available in JSON and CSV format via REST APIs. The Quantmod and the alphavantager packages offer a lightweight R interfaces to the Alpha Vantage API.

getSymbols('AAPL', src="av", output.size="full", adjusted=TRUE, api.key='your API key')
plot(AAPL$adjusted)
